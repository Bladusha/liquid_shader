# Portable Download Branch

This branch contains a full Unity project snapshot stored as regular Git files so it can be downloaded from GitHub with `Download ZIP`.

How to use:
1. Open this branch on GitHub.
2. Click `Code` -> `Download ZIP`.
3. Extract the archive.
4. Open the project in Unity Hub.
5. Recommended Unity version: `6000.4.10f1`.

This branch is for convenient handoff/downloads and is not intended to be used for normal development.
